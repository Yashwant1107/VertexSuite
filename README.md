# Vertex Suite – Modern SaaS Platform Redesign

## Overview

Vertex Suite is a modern SaaS platform redesign built using React, Vite, GSAP, Three.js, and Tailwind CSS. The project reimagines the existing Vertex Suite website with immersive animations, scroll-based storytelling, parallax effects, and interactive 3D experiences.

The redesign focuses on creating a premium user experience that visually represents Vertex Suite's ecosystem of business solutions, including Social Commerce, Marketplace Integration, Shared Inbox, Marketing Automation, Logistics, and Vendor Management.

## Design Concept

The project follows a **Digital Gravity Ecosystem** concept where Vertex Suite acts as the central hub connecting multiple business platforms such as WhatsApp, Shopify, Amazon, WooCommerce, Facebook, Instagram, and logistics partners.

Through interactive animations and visual storytelling, users can experience how Vertex Suite unifies multiple business operations into a single platform.

## Installation

Clone the repository:

```bash
git clone https://github.com/Yashwant1107/VertexSuite.git
```

Navigate to the project directory:

```bash
cd VertexSuite
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
